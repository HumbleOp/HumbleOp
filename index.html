HumbleOp

Debate with purpose. No noise, just ideas.

Why HumbleOp?

  • Clarity wins, not popularity: One comment per user, fair voting, no spam.
  • Real-time voting: One vote per user. Live tallies. Transparent results.
  • Duel system: Top comment enters a 1-on-1 timed exchange with the poster.
  • Respect by logic: Flags and likes affect who holds the stage.
  • Ethical by design: Argon2id authentication, no surveillance.

Who is it for?

HumbleOp is built for:

  • Creators tired of engagement hacks
  • Thinkers who crave structured debate
  • Communities that value respectful dialogue

Try it (MVP)

$ git clone https://github.com/HumbleOp/HumbleOp.git
$ cd HumbleOp
$ python -m venv venv
$ source venv/bin/activate
$ pip install -r requirements.txt
$ python humblecore.py

Then open http://localhost:5000
